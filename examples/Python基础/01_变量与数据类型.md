# Python基础 - 第 1 课：变量与数据类型

## 学习目标

学完本节后，你应该能做到：

- 解释变量是什么。
- 区分常见的数据类型。
- 写出简单的变量赋值代码。

## 内容讲解

变量可以理解成「给一份数据贴标签」。比如你把一盒饼干贴上「早餐」标签，下次想找它时，不需要重新描述这盒饼干，只要说「早餐」就行。

在 Python 里也是这样：

```python
name = "小明"
age = 18
is_student = True
```

这里有三个变量：

| 变量名 | 数据 | 类型 |
| --- | --- | --- |
| `name` | `"小明"` | 字符串 |
| `age` | `18` | 整数 |
| `is_student` | `True` | 布尔值 |

常见数据类型包括：

- 字符串：表示文字，例如 `"hello"`。
- 整数：表示没有小数的数字，例如 `18`。
- 浮点数：表示带小数的数字，例如 `3.14`。
- 布尔值：表示真假，例如 `True` 或 `False`。

## 小结

- 变量是数据的名字。
- `=` 表示把右边的数据赋给左边的变量名。
- 不同类型的数据适合表达不同信息。
- Python 会根据你写入的数据自动识别基本类型。
